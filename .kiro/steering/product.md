# Product

This repository hosts **Kiro Powers** developed by **SDS/TRE-PR** (Tribunal Regional Eleitoral do Paraná).

A "Power" is a packaged integration that bundles documentation, MCP server configuration, and supporting assets (certificates, steering files) to extend the Kiro AI development environment with external service capabilities.

## Current Powers

- **gitlab-trepr** — Integration with the TRE-PR internal GitLab instance (`gitlab.tre-pr.jus.br`). Enables the AI agent to manage projects, merge requests, issues, CI/CD pipelines, wiki pages, releases, milestones, and more via the GitLab REST API.

## Audience

Developers at TRE-PR who use Kiro as their AI-assisted development environment and need seamless access to internal infrastructure services.
